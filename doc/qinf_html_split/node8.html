7.1.2 ketp, brap, braketp

ketp() returns true if the expression e is a ket (column vector). brap() returns true if the expression e is a bra (row vector). braketp() returns (ketp(e) or brap(e)), ie returns true if e is either a ket or a bra.

7.1.3 square_matp

square_matp() returns true if e is a square matrix.
